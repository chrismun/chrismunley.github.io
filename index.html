Welcome

Welcome to our site about programming. Our goal is for you to be able to go out into the real world and make real applications by the time you have gone through our page. Read through our tutorials, practice replicating code examples, try our practice problems, or simply skim the pages. You are sure to learn something! Enjoy!

Main points, important concepts, and more will appear in blue boxes
Useful tips will appear in a green box

Code example:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello World!");
    }
}
The yellow box... is a wildcard. Watch out!
Warning! Watch out for red boxes, as they will steer you away from making simple mistakes.

About me

Hello! I am Chris Munley. Aiden Trainer and I started this site to teach our peers about programming languages that they will confront in their school life and possible in the workforce. There is a seperate page for each language. As a side note, this site is still under construction, so please check back soon for more information and learning.
